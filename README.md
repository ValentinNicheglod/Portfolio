## 📎 Descripción del proyecto...
Actualmente el proyecto se presenta en una sola página con secciones, cada componente representa una sección, en un futuro mi idea es crear otras secciones y agregar un blog.

#### Componentes

###### Greeting
Sección de bienvenida.
###### About
Sección personal, con información y un slider de react-slideview con fotos.
###### Skills
Sección de competencias técnicas, iconos de icon8.
###### Projects
Sección contenedora de `ProjectCards`.
###### ProjectCard
Cada card muestra información del proyecto y permite acceder al repositorio de GitHub, una galería con capturas de pantalla y la opción de visitar .
###### Contact
Sección de contacto y footer.
